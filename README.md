Nothing but a simple python project, I made for fun.

just run main.py, and it gonna work. I hope.

## Commands

**game** - *opens the game menu*  
**uranium** - *shows you an image of uranium*  
**help** - *shows you every command and a short description*  
**whoami** - *shows you your username*  
**hi** - *just sey hi. Im sure its not a big deal*  
**time** - *shows you current time*  
**delete** - *deletes your account*  
**date** - *shows you the current date*  
**clear** - *clears the terminal*  
**open** - *the word that you print after, would be printed to terminal*  
**calc** - *opens a simple calculator*
